# CasaFiducia — GitHub Preview Version

This version is prepared for previewing on GitHub Pages **before buying or connecting casafiducia.ch**.

## Important

This package does **not** include the `CNAME` file, so GitHub Pages should publish under:

```txt
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/
```

## If GitHub still redirects to casafiducia.ch

Go to your repository:

1. `Settings`
2. `Pages`
3. Find `Custom domain`
4. Remove `casafiducia.ch`
5. Save
6. Wait a few minutes and refresh the GitHub Pages URL

## Before final publishing on casafiducia.ch

Use the production package or add back:

```txt
CNAME
```

with:

```txt
casafiducia.ch
```

Then restore:
- canonical URL
- sitemap
- Formspree `_next` redirect
- final email address
- legal details in `privacy.html` and `impressum.html`
